# Orders MicroService

```
docker compose up -d
```

## Development pasos

1. Clonar el proyecto
2. Crear un archivo `.env` basado en el archivo `.env.template`
3. Levantar la base de datos con `docker compose up -d`
4. Levantar el proyecto con `npm run start:dev`
